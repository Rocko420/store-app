# PHP MVC Store Application

## Project Summary

This project is a PHP MVC Store Application developed as a multi-week course project. The application simulates a basic online shopping experience where users can browse products, add items to a shopping cart, remove items from the cart, and complete a checkout process.

The project was built using the MVC (Model-View-Controller) architecture pattern to separate application logic, user interface components, and database operations into organized sections. The application uses PHP for backend functionality, MySQL for database management, HTML/CSS for the frontend interface, and PHP sessions for cart management.

Main features of the application include:

- Product catalog connected to a MySQL database
- Shopping cart functionality
- Add and remove item features
- Automatic cart total calculations
- Tax and shipping calculations
- Checkout process
- Responsive and improved user interface styling
- MVC folder structure
- Prepared SQL statements for improved security
- Session management for cart persistence

The application was tested throughout development to verify database connectivity, cart functionality, checkout processing, and responsive design behavior. Final testing and interface improvements were completed during Week 5 of development.
